# design-docs/core-beliefs

Статус: `Scaffold`.

## Implemented (core beliefs)

1. Код — первичный источник истины.
2. Документация должна ускорять выполнение задач, а не дублировать весь код.
3. AGENTS.md — карта, а не длинный мануал.
4. Сложные работы ведутся через execution plans в репозитории.
5. Все критичные знания должны быть repository-local и versioned.
